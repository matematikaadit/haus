Haus

Here Another URL Shortener

  • Github
  • Issues
  • Pull Request

Usage

GET /
Shows this homepage.

GET /<id>
Get url associated with the <id>.

POST /   PARAM: url=<url>
Create random id and assign <url> to it.

POST /<id>    PARAM: url=<url>
Create <id> and assign <url> to it.

Note:

<id> can only contains lowercase alphanumeric character ([a-z0-9]), dash (-), or underscore (_).
